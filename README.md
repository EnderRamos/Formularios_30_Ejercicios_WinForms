# Formularios 30 Ejercicios en C#

Proyecto WinForms (.NET 8) con un menú principal para acceder a 30 formularios.

## Cómo abrir
1. Descomprime el ZIP.
2. Abre `Formularios30Ejercicios.csproj` en Visual Studio.
3. Ejecuta con F5.

## Contenido
- `MenuForm.cs`: menú principal.
- `Exercise1Form.cs` a `Exercise30Form.cs`: ejercicios separados.
- `BaseExerciseForm.cs`: formulario base para estilo y controles.
